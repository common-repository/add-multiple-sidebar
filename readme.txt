Add Multiple Sidebar is Sidebar base plugin in which user just need to assign Multiple Sidebar to show as Site Sidebar
== Description ==

### Overview
Add Multiple Sidebar is allows the end user to add sidebar name, create your sidebar, delete your sidebar.Easily add Multiple your sidebar.

* Add Sidebar Name

* Create Your Sidebar

* Delete Your Sidebar

== Installation ==

= Using The WordPress Sidebar =

1. Navigate to the Add New in the plugins Sidebar
2. Search for Add Multiple Sidebar
3. Click Install Now
4. Activate the plugin on the Plugin Sidebar

= Uploading in WordPress Sidebar =

1. Navigate to the Add New in the plugins Sidebar
2. Navigate to the Upload area
3. Select add-multiple-sidebar.zip from your computer
4. Click Install Now
5. Activate the plugin in the Plugin Sidebar
